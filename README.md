# escape-plus-U

방탈출 카페 탐색, 후기 공유, 커뮤니티 활동을 한곳에서 다루는 Django 기반 웹 프로젝트입니다.  
단순 정보 나열이 아니라 `카페 탐색 + 지도 검색 + 카테고리형 커뮤니티`를 한 흐름으로 묶는 것을 목표로 정리했습니다.

## 프로젝트 한눈에 보기

- 프로젝트명: `escape-plus-U`
- 유형: 개인 포트폴리오 프로젝트
- 목적: 방탈출 이용자 관점에서 필요한 정보 탐색과 후기 공유를 한 서비스에 묶기
- 메인 앱 경로: [`escapeplus/`](./escapeplus)
- 기술 스택: `Python`, `Django`, `SQLite3`, `Bootstrap 5`, `Kakao Login`, `Kakao Maps`

## 기획 의도

방탈출 이용자는 보통 다음 정보를 여러 채널에서 따로 찾게 됩니다.

- 어떤 카페가 있는지
- 어디에 위치해 있는지
- 실제 후기 분위기가 어떤지
- 같이 갈 사람을 구할 수 있는지
- 예약 양도나 꿀팁 같은 커뮤니티 정보가 있는지

이 프로젝트는 이 흐름을 하나의 웹 서비스로 묶는 연습을 목표로 했습니다.  
특히 단순 게시판 프로젝트에서 끝내지 않고, 아래 세 가지를 함께 담는 방향으로 구성했습니다.

- `정보 탐색`: 카페 목록과 테마 수 확인
- `위치 확인`: 카카오 지도 기반 검색
- `커뮤니티`: 후기, 구직, 양도, 질문/건의 등 카테고리형 게시판

## 핵심 기능

### 1. 게시판

- 카테고리별 게시판 분리
- 게시글 작성, 목록 조회, 상세 조회, 삭제
- 태그 표시
- 페이지네이션
- 실제 화면 확인이 가능하도록 카테고리별 더미 데이터 구성

게시판 카테고리:

- `review`: 리뷰
- `job`: 구직
- `assignment`: 양도
- `make`: 창작
- `suggest`: 질문/건의
- `tip`: 꿀팁
- `team`: 대외활동
- `play`: 온라인

### 2. 회원 기능

- 일반 회원가입
- 로그인 / 로그아웃
- 카카오 로그인 흐름 구성

### 3. 카페 정보

- 카페 목록 조회
- 카페 상세 페이지 조회
- 홈 화면에서 카페 수, 테마 수 요약 표시

### 4. 지도 기능

- 카카오 지도 기반 검색 화면
- 지도 베스트 페이지 구성

## 담당 기능

## 트러블슈팅

### 1. 게시글 수가 적어 페이지네이션 검증이 어려웠던 문제

문제:

- 총 게시글 수만 조금 늘려서는 각 카테고리 화면에서 페이지네이션이 제대로 보이지 않았습니다.
- 카테고리형 게시판은 `전체 게시글 수`보다 `카테고리별 게시글 수`가 더 중요했습니다.

해결:

- 게시판 카테고리별로 각각 20개씩 데이터가 보이도록 시드 데이터를 보강했습니다.
- 덕분에 모든 카테고리에서 목록 2페이지 기준 UI를 바로 검증할 수 있게 정리했습니다.

### 2. 작성자명이 모두 `테스트유저`로 보여 더미 데이터 티가 강했던 문제

문제:

- 게시글 수만 늘려도 작성자명이 하나로 고정되면 실제 커뮤니티처럼 보이지 않았습니다.

해결:

- 여러 닉네임 기반 계정을 추가하고 게시글 작성자를 분산했습니다.
- 목록 화면에서 `한 사람이 도배한 게시판`처럼 보이던 인상을 줄였습니다.

### 3. 페이지네이션 활성 상태가 파란색 계열과 겹쳐 가독성이 떨어진 문제

문제:

- Bootstrap 기본 페이지네이션 스타일은 현재 색상 톤과 겹쳐 활성 페이지가 또렷하게 보이지 않았습니다.

해결:

- 페이지네이션 버튼을 카드형으로 바꾸고, 활성/비활성/호버 상태를 별도 스타일로 분리했습니다.
- 이전/다음 버튼도 아이콘형으로 정리해 탐색 흐름이 더 선명하게 보이도록 수정했습니다.

## 실행 방법

### 1. 가상환경 활성화

```bash
source .venv/bin/activate
```

### 2. 의존성 설치

```bash
pip install -r requirements.txt
```

### 3. 서버 실행

```bash
cd escapeplus
python manage.py runserver
```

접속 주소:

```text
http://127.0.0.1:8000/
```

## 사용 데이터 / 개발 환경

- DB 파일: [`escapeplus/db.sqlite3`](./escapeplus/db.sqlite3)

### 데이터 안내

본 프로젝트에 포함된 방탈출 카페/테마 데이터는 **포트폴리오 시연 목적의 샘플 데이터**입니다.

- **카페명, 테마명, 주소**: 공개된 사실 정보를 기반으로 정리
- **평점, 리뷰 수, 탈출률**: 시연용으로 생성한 가상의 데이터이며 실제 수치가 아닙니다
- **가격, 공포도, 활동성**: 업계 평균과 장르 특성을 참고한 추정값입니다

실제 서비스 운영 시에는 각 카페와의 제휴 또는 사용자 리뷰 기반으로 데이터를 구성해야 합니다.
